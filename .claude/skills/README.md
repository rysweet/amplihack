# Claude Code Skills

This directory contains Claude Code skills for amplihack. Skills are modular, reusable capabilities that load on-demand.

## Skill Structure

Each skill follows this pattern:

```
skill-name/
├── SKILL.md         # Core skill definition (required)
├── README.md        # User-facing documentation (optional)
├── examples/        # Usage examples (optional)
└── tests/           # Skill tests (optional)
```

## Available Skills

For the complete skill catalog with descriptions and usage, see:
**[docs/skills/SKILL_CATALOG.md](../../docs/skills/SKILL_CATALOG.md)**

## Creating New Skills

Use `/amplihack:skill-builder` to create new skills following amplihack philosophy.
