# Recipe Step Executor — Gherkin Specification

## Goal

Implement a Python `RecipeStepExecutor` class that satisfies every scenario in the attached `.feature` specification. The feature file is the sole specification — implement exactly what the scenarios describe.

## Deliverables

1. `RecipeStepExecutor` class with an `execute(recipe, context)` method.
2. Focused tests covering each scenario from the feature file and cross-feature interactions.

## Non-goals

- Do not implement a CLI or REST API.
- Do not implement persistent storage.
- Do not implement distributed execution.
- Do not implement a recipe parser beyond dict/JSON input.
