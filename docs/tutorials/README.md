# Tutorials

Learn amplihack step-by-step with hands-on tutorials.

## What Are Tutorials?

Tutorials are **learning-oriented** guides that teach you amplihack through practical exercises. They're designed to help you build understanding by doing, not just reading.

Following the [Diataxis framework](https://diataxis.fr/), tutorials differ from other documentation:

- **Tutorials**: Learning by doing (you are here)
- **How-To Guides**: Task-focused solutions ([../howto/](../howto/))
- **Reference**: Complete technical information ([../reference/](../reference/))
- **Explanation**: Concepts and background ([../ concepts/](../claude/context/))

## Tutorial Format

All tutorials follow this structure:

1. **What You'll Learn** - Clear learning objectives
2. **Prerequisites** - What you need before starting
3. **Step-by-Step Instructions** - Numbered steps with commands
4. **Expected Outcomes** - What success looks like
5. **Troubleshooting** - Common issues and solutions
6. **Next Steps** - Where to go after completing
