# Goal: API Documentation Generator

Automatically generate comprehensive API documentation from code, following OpenAPI/Swagger standards.

## Constraints

- Must complete within 20 minutes
- Should generate OpenAPI 3.0 spec
- Must include examples for all endpoints
- Should detect authentication requirements

## Success Criteria

- Generates complete OpenAPI specification
- Documents all REST endpoints (GET, POST, PUT, DELETE)
- Includes request/response schemas
- Provides code examples in multiple languages
- Documents authentication and authorization
- Generates interactive API documentation

## Context

Create API documentation by:

- Analyzing FastAPI/Flask route decorators
- Extracting request/response models
- Generating OpenAPI spec
- Creating example requests
- Documenting error responses
