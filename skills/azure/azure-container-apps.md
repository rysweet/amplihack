# Azure Container Apps

## Key Features
- Serverless container hosting
- Built-in Dapr support
- Auto-scaling (including to zero)
- Revision management

## Quick Reference
```bash
# Create container app
az containerapp create --name myapp --resource-group rg --environment myenv --image myregistry.azurecr.io/myimage:latest
```
