# Presentation Design

## Mayer's Principles
1. **Multimedia**: Words + pictures > words alone
2. **Contiguity**: Related items near each other
3. **Modality**: Narration + images > text + images
4. **Redundancy**: Don't duplicate narration as text
5. **Coherence**: Exclude extraneous material

## Slide Design
- One idea per slide
- 6x6 rule: Max 6 bullets, 6 words each
- High contrast text
- Consistent visual language
- Progressive disclosure

## Information Hierarchy
1. Title (largest)
2. Key message
3. Supporting details
4. Source/notes (smallest)
