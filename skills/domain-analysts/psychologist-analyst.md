# Psychologist Analyst

Apply psychological insights to understand human behavior, cognition, and motivation.

## When to Use

- User behavior analysis
- Cognitive load assessment
- Motivation and engagement
- Keywords: "behavior", "motivation", "cognitive", "bias", "user", "mental model"

## Theoretical Foundations

- **Cognitive Psychology**: Mental processes (attention, memory, decision-making)
- **Behavioral Psychology**: Observable behavior and reinforcement
- **Social Psychology**: Group dynamics and influence
- **Developmental Psychology**: Growth and change over time

## Key Frameworks

### 1. Cognitive Bias Checklist
| Bias | Description | Mitigation |
|------|-------------|------------|
| Confirmation | Seek confirming evidence | Devil's advocate |
| Anchoring | Over-rely on first info | Consider alternatives |
| Availability | Recent = likely | Base rates |
| Sunk cost | Can't let go | Focus on future value |

### 2. Motivation Analysis (Self-Determination Theory)
- Autonomy: Control over actions
- Competence: Mastery and effectiveness
- Relatedness: Connection to others

### 3. Cognitive Load Theory
- Intrinsic load: Complexity of material
- Extraneous load: Poor design
- Germane load: Schema building
- Goal: Minimize extraneous, optimize germane

### 4. Behavior Change Model
```
Behavior = Motivation × Ability × Prompt
```

## Core Questions

1. What motivates this behavior?
2. What cognitive biases might be at play?
3. What's the cognitive load?
4. What mental models do users have?
5. What emotions are involved?
6. How do social factors influence behavior?
7. What reinforcement patterns exist?
8. What barriers prevent desired behavior?
